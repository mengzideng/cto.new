# 切西瓜网页小游戏 🍉

一个类似水果忍者的切西瓜网页小游戏，使用纯 HTML5、CSS3 和 JavaScript 开发。

## 游戏特性

### 核心功能
- ✅ 西瓜从屏幕底部随机位置抛出，呈现真实的抛物线轨迹
- ✅ 用户通过鼠标滑动或触摸屏幕来"切"西瓜
- ✅ 切中西瓜后显示切开动画效果，西瓜分成两半并掉落
- ✅ 实时显示分数，每切中一个西瓜得 10 分
- ✅ 如果西瓜掉落未被切中，扣除生命值（共 3 条命）
- ✅ 游戏结束后显示最终分数和重新开始按钮

### 技术实现
- ✅ HTML5 Canvas 进行图形渲染
- ✅ 纯 JavaScript 实现游戏逻辑（无框架依赖）
- ✅ CSS3 实现页面布局和样式
- ✅ requestAnimationFrame 实现流畅动画（60 FPS）
- ✅ 鼠标移动轨迹效果（切割线）

### UI/UX 特性
- ✅ 简洁清爽的游戏界面
- ✅ 明确的分数和生命值显示
- ✅ 流畅的切割反馈动画
- ✅ 响应式设计，适配不同屏幕尺寸
- ✅ 完整的开始菜单和游戏结束界面
- ✅ 移动端触摸支持

### 游戏机制
- 🎮 物理引擎：真实的重力模拟和抛物线轨迹
- 📈 难度递增：每 50 分西瓜生成速度加快
- 🎯 碰撞检测：精确的线段与圆形碰撞检测算法
- ✨ 粒子效果：西瓜被切开后的两半飞溅效果

## 如何游玩

1. 打开 `index.html` 文件（可以直接在浏览器中打开）
2. 点击"开始游戏"按钮
3. 用鼠标滑动或在触摸屏上滑动来切西瓜
4. 尽量不要让西瓜掉落，保护你的生命值
5. 挑战更高的分数！

## 游戏规则

- **得分**：每切中一个西瓜得 10 分
- **生命值**：初始 3 条生命
- **失误**：西瓜掉落到屏幕底部扣除 1 条生命
- **游戏结束**：生命值耗尽时游戏结束
- **难度递增**：随着分数增加，西瓜生成速度会加快
